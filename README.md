## Project Description

This project implements the Bucket Sort algorithm without using any built-in sorting functions.  
The program sorts a list of integers and tracks the number of comparisons and swaps.

Experiments were conducted on different input sizes to observe the algorithm's performance.

## Output

The program records:

- Sorted list
- Execution time
- Number of comparisons
- Number of swaps

Results are stored in the **results** folder.

## Algorithm Used

Bucket Sort with Insertion Sort to sort elements within each bucket.

## Time Complexity

Best Case: O(n)  
Average Case: O(n)  
Worst Case: O(n²)

## Space Complexity

O(n + k)
